The Simple Mirror is a solid conductor material that could be used for simple
props. It uses a fixed roughness value that mimics perfectly sharp mirrors.

Inputs

- Object Info 🢒 Color and Alpha
    Shader's base color and transparency are influenced by object's Viewport
    Display Color.

Outputs

- Cycles and EEVEE BSDF
    Streamlined material output that favours speed over quality.

- Fallback Principled BSDF
    For different render engines and GLTF export.

    Unfortunately, Object Info is not implemented in Hydra Storm, resulting in
    a completely transparent material.
